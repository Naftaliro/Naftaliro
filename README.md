# Hey, I'm Naftali

I build open-source tools for Linux desktop users — the kind of things that should exist but don't. My focus is on making system configuration accessible to people who shouldn't need a terminal to get their computer working the way they want.

## What I'm Working On

| Project | Description |
|---------|-------------|
| [**First Steps**](https://github.com/Naftaliro/first-steps) | A GTK4 + LibAdwaita onboarding wizard for Zorin OS and Ubuntu-based systems. Handles codecs, Flatpak, drivers, backups, power profiles, firewall, and more — all through a clean, checkbox-driven UI. Ships as a `.deb` package with auto-update. |
| [**GNOME Theme Switcher**](https://github.com/Naftaliro/gnome-theme-switcher) | A terminal-based TUI for switching between system-wide GNOME themes. Zero dependencies, 9 built-in themes, custom theme support. |
| [**Zorin OS GNOME Themes**](https://github.com/Naftaliro/zorinos-gnome-themes) | System-wide GNOME theme install scripts for Zorin OS 18 and Ubuntu. macOS and Windows-inspired themes with one-liner installers. |

## What I Care About

I think Linux desktops are close to being genuinely great for everyday users, but the last mile of polish is often missing. My projects try to fill those gaps — proper onboarding, easy theming, sensible defaults — without dumbing anything down.

Everything I ship is open-source (**GPL-3.0** for First Steps, **MIT** for the theme tools), properly audited, and designed to be installed and removed cleanly.

## Tech Stack

**Languages:** Python, Bash

**Frameworks:** GTK4, LibAdwaita, PyGObject

**Platforms:** Zorin OS, Ubuntu, Debian-based Linux

**Packaging:** `.deb`, polkit, AppStream, GitHub Actions CI/CD
